# Amharic Learning Dataset & Explorer | مشروع تعلم اللغة الأمهرية

هذا المشروع يحتوي على مجموعة بيانات شاملة لتعلم اللغة الأمهرية، تضم أكثر من 5000 جملة وكلمة مترجمة إلى العربية والإنجليزية، مع أدوات استعراض وبحث تفاعلية.

This project contains a comprehensive dataset for learning Amharic, featuring over 5,000 sentences and words translated into Arabic and English, along with interactive exploration and search tools.

## محتويات المشروع | Project Contents

- **Data Files**: 
  - `amharic_5000_corrected_en.xlsx`: الملف الكامل مع الترجمة الإنجليزية.
  - `amharic_data.json` & `sentences_data.json`: البيانات بصيغة JSON للاستخدام البرمجي.
  - `amharic_5000_corrected.csv`: نسخة CSV من البيانات.
- **Interactive Tools**:
  - `amharic_sentence_explorer_final.html`: مستعرض الجمل التفاعلي.
  - `amharic_search.html`: أداة البحث في الكلمات والجمل.
- **Documentation**:
  - `amharic_5000_summary.pdf`: ملخص إحصائي للمشروع.

## كيفية التشغيل | How to Run

1. قم بتحميل المستودع أو عمل Clone له.
2. لفتح أدوات الاستعراض، ببساطة افتح ملفات الـ HTML (مثل `amharic_sentence_explorer_final.html`) في أي متصفح ويب حديث.
3. لاستخدام البيانات في مشاريعك، يمكنك الاعتماد على ملفات الـ JSON أو CSV.

1. Download or clone the repository.
2. To use the exploration tools, simply open the HTML files (e.g., `amharic_sentence_explorer_final.html`) in any modern web browser.
3. To use the data in your projects, you can rely on the JSON or CSV files.

## المتطلبات | Requirements

- متصفح ويب حديث (Chrome, Firefox, Edge, Safari).
- (اختياري) Python 3.x إذا كنت ترغب في معالجة البيانات برمجياً.

- Modern web browser.
- (Optional) Python 3.x for programmatic data processing.
